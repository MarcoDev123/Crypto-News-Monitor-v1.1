# Crypto News Monitor v1.1
Crypto News Monitor keeps traders informed with live crypto headlines, BTC candlestick charts, coin and macro tickers, fear and greed data, dominance metrics, and market alerts. It filters factual, market-moving events so you can track crypto, stocks, Fed news, oil, and influencer posts in one clean desktop dashboard.
